## Cade Information

Cade is a discord music bot. Thats it. Nothing special.
## Commands

The commands are simple. We legit use / commands as discord are going to remove prefix commands. See, we are keeping up. :)

```
/play
/queue
/pause
/resume
/nowplaying
/skip
/join
/leave
/loop
/volume 

**New Commands**
/stop
```
